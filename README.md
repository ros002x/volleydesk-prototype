# VolleyDesk Prototype

Prototype statico HTML/CSS/JavaScript per gestione atleti, certificati, contabilita, documenti e deposito dati di una societa sportiva.

## Funzioni principali

- Dashboard premium responsive
- Elenco atleti con modale modifica
- Certificati medici con stati colore
- Contabilita stagione settembre-agosto
- Selezione multipla e cancellazione
- Archivio documenti e anteprima dati JSON

## Avvio

Apri `index.html` nel browser.
